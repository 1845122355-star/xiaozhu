## 第一篇 发改委工作全景

### 第1章 发改委的定位与职责
- 发改委是地方政府宏观经济管理与综合协调的职能部门，承担全区经济平衡、重大项目统筹、产业政策、价格管理、投融资管理、区域发展等职责。  
- 核心职责列表：
  1. 编制并组织实施国民经济和社会发展规划（中长期与专项规划）。
  2. 组织和指导重大项目的立项、可行性研究、建设推进和绩效评估。
  3. 管理城乡及区域协调发展，推动产业结构调整与要素保障。
  4. 指导财政性投资与专项债管理，协调投融资体制改革。
  5. 参与或牵头重大行政许可与联合审批工作。
  6. 负责涉军、涉保、重大安全相关项目的审查和报批对接。

### 第2章 机关组织与处室分工（地方特色）
- 机关架构概览（处室职能精要）：
  - 办公室：综合协调、文电、会务、保密与印章管理。
  - 规划处：中长期规划编制与政策研究。
  - 投资处：政府投资项目管理、资金来源与专项债管理。
  - 项目监管处 / 投资监管科：项目台账、进度与绩效考核。
  - 产业处：产业政策、园区建设与产业链对接。
  - 价格与市场处：价格监测、涉企收费与市场秩序建议。
  - 经济国防处：涉军涉保审查、国民经济动员。
  - 法制与政策处（法制科）：法规把关、合规审查与行政复议对接。
  - 外联与合作处：对外合作、对接商务与海关、外经贸支持。

### 第3章 协同机制与常见工作流程
- 常见协同方式：联合会审、牵头办会、跨部门会商、责任清单、项目台账对接、专项调度会。  
- 典型流程示例：政府投资项目立项（建议书 → 可研 → 概算 → 立项 → 施工 → 验收）——每一环节的主要材料、处室职责、时间节点与风险点（详列）。

### 第4章 办文与审批链条（要点）
- 材料准备：每个报告或请示必须明确：事项背景、要解决的问题、拟办意见与建议、法律依据、资金来源、风险点、拟请批示或决定。  
- 会签/征求意见：会签单必须列明会签单位、意见摘要、会签负责人、会签时间并在材料中附会签单扫描件或导出 PDF。  
- 档案要求：电子文本采用统一命名（YYYYMMDD_处室_文件简称_v1.md），并在元数据中包含作者、版本、审批链与归档标签。

## 第二篇 宏观经济政策（对发改工作的指导要点）

### 第1章 宏观政策框架
- 经济政策工具（财政、货币、产业、区域与结构性改革）之间的配合原则与在项目评估中的体现。  
- 在编制报告或向领导汇报时务必说明政策背景、上位文件依据与时间窗口（如专项债使用窗口、国家战略抓手）。

### 第2章 财政与债务管理要点
- 项目资金结构表模板（必备字段：资金来源、额度、到位凭证、使用计划、预算安排）。  
- 专项债申报与使用合规性审核要点：用途限定、偿债来源、财政承诺函、地方配套资金证明。

### 第3章 货币、金融与融资
- 项目融资方案要点：融资成本、期限匹配、政府代偿/分担条款、合同履约保障、应急处置条款。  
- 政策性金融工具使用注意：再贷款/再贴现支持条件、国开行/政策性银行合作框架。

### 第4章 产业政策与空间战略
- 产业培育措施（税收、土地、用能、人才激励）具体落地建议。  
- 区域发展与产业承接（北部湾、向海通道、边境口岸经济带）的要点与风险清单。
